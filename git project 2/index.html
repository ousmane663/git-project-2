les etapes a suivre

Avant toute chose aller dans le dossier, et faites clique gauche, open git bash here


1- touch index.html---pour céer ce fichier


2- tu ouvre le fichier avec un editeur de texte, ou tu y inscrit le code souhaité.


3-faire la commande : git init -- pour initialiser le projet en créant un fichier caché


4- identification : on saisie le nom d'utilisateur et l'email de notre compte GITUP en faisant les commandes suivant


git config --global user.name "..." git config --global user.email "..."


5- apres l'identification ajoute ton fichier index.html en faisant le code suivant : git add index.html


git status permet de voir tous les fichier présauvegardé


pour supprimer un fichier présauvegarder utiliser git rm exemple ici : git rm --cached index.html. note : si vous ne mettez pas "--" il va supprimer définitivement le fichier, aulieu de l'enlever uniquement dans la liste des fichiers présauvegarder


créons un nouveau ficher css : touch style.css


Comment ajouter tous les fichier présent dans un dossier : git add. (pour sélectionner tous les fichiers du dossier)


Pour supprimer tous les fichier d'un dossier il faut faire : git rm -r cached.
les fichier dans le dossier idea : git rm -r cached .idea/


Si l'on veut ajouter plusieur fichiers specifiques par exemple ici tous les fichiers . html et . css on fait : git add *.html *.css


la commande checkout permet de retourner en arriere ex : git checkout --. (ici le point signifie tous les fichier)
on utilise git commit pour sauvegarder les fichiers ou dossier ex : git commit.
on utilise git commit -m pour mettre le commentaire sur le changement ex : git commit -m 'changement html et css'


6- git branch
faire : git status (pour savoir dans quelle branche on se trouve ex: branch master; pour créer une branche il faut utiliser git branch ... ex : git branch Mon-dossier (cela va créer une branche Mon-dossier).
Pour changer de branche, on utilise : git chekout ... ex : git checkout Mon-dossier (cela vous connecte à la branche Mon-dossier).
si vous oubliez les des branches que vous avez créer, trouvez les avec : git branch --list.
Créeons un fichier js dans la branche Mon-dossier : touch main.js

retournons à la branche Mon-dossier, puis ajoutons nos fichiers : git branch Mon-dossier; git add .
maintenant changeons de branche : git checkout master ; puis fusionnons les 2 branches master et Mon-dossier : git merge Mon-dossier


maintenant connectons notre travail local avec GitHub, pour ce faire ex : git remote add origin https://github.com/ousmane663/nom-repertoire.git
git push -u origin master


maintenant on est connecté à github, tous nos fichiers y sont importés, créons un readme.md et sauvegardons le dans le git bash : touch README.md, git add. ; git commit -m "ajout de README" , maintenant importons le ou poussons le dans le github : git push


si il y a d'autres développeur qui ont apportés des changement dans le projet, pour récupérer cela, utiliiser git pull ex git pull
si l'on perd ses dossiers en local, on peut les récupérer en utilisant git clone et en y rattachant l'adresse qu'on recuper dans dowlod sur github ex : git clone https://github/ousmane663/repertoire.git (cela va télécharger vos documents enregistré dans github
